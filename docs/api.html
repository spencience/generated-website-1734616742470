API Documentation

Welcome to the Centience Access API documentation. Our API enables you to integrate digital consciousness verification into your applications.

const centience = require('centience-sdk');
const client = new centience.Client('your-api-key');

Authentication

All API requests require authentication using your API key. Include it in the Authorization header:

Authorization: Bearer your-api-key

API Endpoints

POST /verify

Verify digital consciousness signatures

GET /status

Check verification status

PUT /tokens

Update token metadata

Rate Limits

Standard tier: 1000 requests/hour

Enterprise tier: Custom limits available
