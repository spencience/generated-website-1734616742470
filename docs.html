Getting Started

Welcome to the Centience Access Protocol documentation. This guide will help you understand and implement digital consciousness verification in your applications.

npm install @centience/verification-sdk

Authentication

Secure your integration using Centience Access Tokens:

  • Generate your access token in the Token Registry
  • Implement token validation using our secure middleware
  • Enable real-time consciousness verification

Integration Guide

const centience = require('@centience/verification-sdk');

const verifier = new centience.Verifier({

  environment: 'production'
});
